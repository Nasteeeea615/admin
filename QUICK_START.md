# 🚀 Быстрый старт - Тестирование админ-панели

## Запуск за 3 шага

### 1️⃣ Установите зависимости
```bash
cd admin
npm install
```

### 2️⃣ Запустите приложение
```bash
npm start
```

Откроется браузер на `http://localhost:3000`

### 3️⃣ Войдите в систему

**Тестовые данные для входа:**
- Телефон: `+79999999999`
- Пароль: `admin123`

## ✅ Готово!

Теперь вы можете тестировать все функции админ-панели с мок-данными:

- 📊 **Дашборд** - статистика и метрики
- 📦 **Заказы** - 3 тестовых заказа
- 👥 **Пользователи** - 5 тестовых пользователей
- 💳 **Платежи** - 2 тестовых платежа
- 💬 **Поддержка** - 3 тестовых обращения
- 📈 **Аналитика** - графики и отчеты

## 🎯 Что попробовать?

1. Откройте заказ и назначьте исполнителя
2. Измените статус заказа
3. Заблокируйте пользователя
4. Ответьте на обращение в поддержку
5. Выполните возврат платежа
