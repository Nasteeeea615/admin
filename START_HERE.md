# 🎉 Добро пожаловать в админ-панель!

## Готово к тестированию! 🚀

Я настроил админ-панель для работы с **мок-данными**, так что вы можете протестировать её прямо сейчас, без запуска backend.

---

## Запуск (2 команды)

```bash
cd admin
npm install
npm start
```

---

## Вход в систему

После запуска откроется браузер. Используйте:

```
Телефон: +79999999999
Пароль:  admin123
```

---

## Что внутри?

✅ **3 заказа** - с разными статусами (ожидает, в работе, выполнен)  
✅ **5 пользователей** - клиенты и исполнители  
✅ **2 платежа** - завершенный и в ожидании  
✅ **3 обращения** - в поддержку с разными статусами  
✅ **Аналитика** - графики и статистика  

---

## Что можно делать?

- 👀 Просматривать заказы и их детали
- 🔄 Менять статусы заказов
- 👤 Назначать исполнителей
- 🚫 Блокировать пользователей
- ✅ Верифицировать исполнителей
- 💰 Обрабатывать платежи и возвраты
- 💬 Отвечать на обращения в поддержку
- 📊 Смотреть аналитику
